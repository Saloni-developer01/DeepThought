A dynamic, responsive web application built as part of the DeepThought selection process. This project replicates a specific Figma design and dynamically renders data from a JSON API.

---

## üöÄ Features

- **Dynamic Content Rendering:** Data is fetched from the DeepThought JSON API.
- **CORS Fallback Mechanism:** Includes a robust error-handling system that uses local data if the API is unreachable due to CORS restrictions.
- **Interactive Sidebar (Journey Board):** A collapsible sidebar that expands to show project assets.
- **Notice Board:** A sliding vertical notice board on the right side, matching the Figma design.
- **Responsive Components:**
  - Video asset with custom thumbnail and click-to-play functionality.
  - Interactive Threadbuild section with collapsible sub-threads.
  - Structured Pointers/Article editor layout.
  - 4SA Method informational cards.
